# go-devgraph

Auto-generated Go client for the Devgraph API.

## Installation

```bash
go get github.com/arctir/go-devgraph
```

## Usage

```go
import "github.com/arctir/go-devgraph/pkg/apis/devgraph/v1"

// Client usage examples will be available after first generation
```

## Auto-Generation

This client is automatically regenerated when the Devgraph API specification changes.

The client is generated using [ogen](https://github.com/ogen-go/ogen) from the OpenAPI spec.
